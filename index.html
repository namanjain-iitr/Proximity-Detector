Proximity Detection using WiFi

Introduction

This project detects the proximity of a device based on its MAC address. It makes use of WiFi signals to determine the signal strength of the device and estimate its distance from the source.

Hardware Requirements

  • ESP8266 WiFi module
  • RGB Led

Software Requirements

  • Arduino IDE
  • ESP8266WiFi Library
  • ArduinoJson library
